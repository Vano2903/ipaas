IPAAS

Introduction

IPaaS (Itis Paleocapa as a Service) is a PaaS for the itis paleocapa and will let the students create databases and web based applications (for now) on the school's servers.

The idea was born from the inability to use AWS for learning purposes due to the requirement of a credit card to use the service.

We decided to create this application to separate the school from others cloud providers such as altervista or heroku.

Authentication

The students will be able to authenticate only using PaleoID which is a google oauth2.0 wrapper created by Cristian Livella that let only the @itispaleocapa.it domain authenticate.
